# 📱 Deep Linking Setup для PingPoint Driver App

## 🎯 Цель

Когда брокер создаёт груз и генерирует ссылку типа:

```
pingpoint://driver/drv_a0a6cf89f39b3b3...
```

При клике на эту ссылку на телефоне должно:

1. Открыться приложение PingPoint Driver
1. Показать детали груза
1. Позволить водителю принять груз и начать трекинг

-----

## ✅ ЧТО УЖЕ НАСТРОЕНО

### В `app.json`:

- ✅ `"scheme": "pingpoint"` - указано что приложение обрабатывает `pingpoint://`
- ✅ iOS `associatedDomains` - для https://pingpoint.suverse.io
- ✅ Android `intentFilters` - для pingpoint:// и https://pingpoint.suverse.io

### В `App.tsx`:

- ✅ `LinkingOptions` настроены с prefixes для `pingpoint://`
- ✅ Navigation Container получает linking конфигурацию

### В `DashboardScreen.tsx`:

- ✅ Экран получает `token` из route params и обрабатывает его

-----

## 🔧 ЧТО НУЖНО ДОБАВИТЬ

### 1. Обновить Deep Linking Config в App.tsx

Нужно добавить поддержку для открытия конкретного груза.

**Файл:** `client/App.tsx`

Заменить эту часть:

```javascript
const linking: LinkingOptions<RootStackParamList> = {
  prefixes: [
    Linking.createURL("/"),
    "pingpoint://",
    "https://pingpoint.app",
    "https://pingpoint.suverse.io",
  ],
  config: {
    screens: {
      Main: {
        screens: {
          Dashboard: "driver/:token",
        },
      },
    },
  },
};
```

На эту:

```javascript
const linking: LinkingOptions<RootStackParamList> = {
  prefixes: [
    Linking.createURL("/"),
    "pingpoint://",
    "https://pingpoint.app",
    "https://pingpoint.suverse.io",
  ],
  config: {
    screens: {
      Main: {
        screens: {
          Dashboard: {
            path: "driver/:token",
            parse: {
              token: (string) => string,
            },
            stringify: {
              token: (token) => token,
            },
          },
          // Новый маршрут для грузов
          LoadDetails: {
            path: "loads/:loadId",
            parse: {
              loadId: (string) => string,
            },
            stringify: {
              loadId: (loadId) => loadId,
            },
          },
        },
      },
    },
  },
};
```

### 2. Добавить новый экран LoadDetailsScreen

**Файл:** `client/screens/LoadDetailsScreen.tsx`

```typescript
import React, { useEffect, useState } from "react";
import { View, StyleSheet, ScrollView, ActivityIndicator, Pressable } from "react-native";
import { useRoute, useNavigation } from "@react-navigation/native";
import { useSafeAreaInsets } from "react-native-safe-area-context";
import { Feather } from "@expo/vector-icons";

import { ThemedText } from "@/components/ThemedText";
import { PingPointColors, Spacing, BorderRadius, Typography } from "@/constants/theme";

interface Load {
  id: string;
  origin: string;
  destination: string;
  rate: number;
  commodity: string;
  weight: number;
  equipmentType: string;
  brokerName: string;
  brokerPhone?: string;
  brokerEmail?: string;
  pickupTime?: string;
  deliveryTime?: string;
  status: "available" | "accepted" | "in_transit" | "completed";
}

type LoadDetailsRouteProp = {
  params: {
    loadId: string;
  };
};

export default function LoadDetailsScreen() {
  const route = useRoute<LoadDetailsRouteProp>();
  const navigation = useNavigation();
  const insets = useSafeAreaInsets();
  
  const [load, setLoad] = useState<Load | null>(null);
  const [isLoading, setIsLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);
  const [isAccepting, setIsAccepting] = useState(false);

  const { loadId } = route.params || {};

  useEffect(() => {
    if (!loadId) {
      setError("No load ID provided");
      setIsLoading(false);
      return;
    }

    const fetchLoad = async () => {
      try {
        setIsLoading(true);
        setError(null);

        // Запрашиваем данные груза с production сервера
        const response = await fetch(
          `https://pingpoint.suverse.io/api/loads/${loadId}`
        );

        if (!response.ok) {
          throw new Error(`Failed to fetch load: ${response.statusText}`);
        }

        const data = await response.json();
        setLoad(data);
      } catch (err) {
        setError(err instanceof Error ? err.message : "Unknown error");
        console.error("[LoadDetails] Error fetching load:", err);
      } finally {
        setIsLoading(false);
      }
    };

    fetchLoad();
  }, [loadId]);

  const handleAcceptLoad = async () => {
    if (!load) return;

    try {
      setIsAccepting(true);

      // Отправляем запрос о принятии груза
      const response = await fetch(
        `https://pingpoint.suverse.io/api/loads/${load.id}/accept`,
        {
          method: "POST",
          headers: {
            "Content-Type": "application/json",
          },
          body: JSON.stringify({
            driverId: "current_driver_id", // Нужно получить из контекста водителя
          }),
        }
      );

      if (!response.ok) {
        throw new Error("Failed to accept load");
      }

      // Навигируем на Dashboard с active load
      navigation.navigate("Main" as never, {
        screen: "Dashboard",
      } as never);
    } catch (err) {
      setError(err instanceof Error ? err.message : "Failed to accept load");
    } finally {
      setIsAccepting(false);
    }
  };

  if (isLoading) {
    return (
      <View style={[styles.container, styles.centerContent]}>
        <ActivityIndicator size="large" color={PingPointColors.cyan} />
      </View>
    );
  }

  if (error || !load) {
    return (
      <View style={[styles.container, styles.centerContent]}>
        <Feather name="alert-circle" size={48} color={PingPointColors.error} />
        <ThemedText style={styles.errorText}>
          {error || "Load not found"}
        </ThemedText>
        <Pressable
          style={styles.backButton}
          onPress={() => navigation.goBack()}
        >
          <ThemedText style={styles.backButtonText}>Go Back</ThemedText>
        </Pressable>
      </View>
    );
  }

  return (
    <View style={[styles.container, { paddingTop: insets.top }]}>
      <ScrollView
        style={styles.content}
        contentContainerStyle={styles.contentContainer}
        showsVerticalScrollIndicator={false}
      >
        {/* Маршрут */}
        <View style={styles.routeCard}>
          <View style={styles.locationRow}>
            <View style={styles.locationIcon}>
              <Feather name="map-pin" size={24} color={PingPointColors.success} />
            </View>
            <View style={styles.locationInfo}>
              <ThemedText style={styles.label}>Pickup</ThemedText>
              <ThemedText style={styles.locationText}>{load.origin}</ThemedText>
            </View>
          </View>

          <View style={styles.divider} />

          <View style={styles.locationRow}>
            <View style={styles.locationIcon}>
              <Feather name="flag" size={24} color={PingPointColors.error} />
            </View>
            <View style={styles.locationInfo}>
              <ThemedText style={styles.label}>Delivery</ThemedText>
              <ThemedText style={styles.locationText}>
                {load.destination}
              </ThemedText>
            </View>
          </View>
        </View>

        {/* Коммерческая информация */}
        <View style={styles.infoCard}>
          <ThemedText style={styles.cardTitle}>COMMERCIAL INFO</ThemedText>

          <View style={styles.infoRow}>
            <ThemedText style={styles.infoLabel}>Rate</ThemedText>
            <ThemedText style={styles.rateValue}>${load.rate}</ThemedText>
          </View>

          <View style={styles.infoRow}>
            <ThemedText style={styles.infoLabel}>Commodity</ThemedText>
            <ThemedText style={styles.infoValue}>{load.commodity}</ThemedText>
          </View>

          <View style={styles.infoRow}>
            <ThemedText style={styles.infoLabel}>Weight</ThemedText>
            <ThemedText style={styles.infoValue}>
              {(load.weight / 1000).toFixed(1)} tons
            </ThemedText>
          </View>

          <View style={styles.infoRow}>
            <ThemedText style={styles.infoLabel}>Equipment</ThemedText>
            <ThemedText style={styles.infoValue}>{load.equipmentType}</ThemedText>
          </View>
        </View>

        {/* Брокер информация */}
        <View style={styles.infoCard}>
          <ThemedText style={styles.cardTitle}>BROKER</ThemedText>

          <View style={styles.infoRow}>
            <ThemedText style={styles.infoLabel}>Name</ThemedText>
            <ThemedText style={styles.infoValue}>{load.brokerName}</ThemedText>
          </View>

          {load.brokerPhone && (
            <View style={styles.infoRow}>
              <ThemedText style={styles.infoLabel}>Phone</ThemedText>
              <ThemedText style={styles.infoValue}>{load.brokerPhone}</ThemedText>
            </View>
          )}

          {load.brokerEmail && (
            <View style={styles.infoRow}>
              <ThemedText style={styles.infoLabel}>Email</ThemedText>
              <ThemedText style={styles.infoValue}>{load.brokerEmail}</ThemedText>
            </View>
          )}
        </View>
      </ScrollView>

      {/* Accept Button */}
      {load.status === "available" && (
        <Pressable
          style={[
            styles.acceptButton,
            isAccepting && styles.acceptButtonDisabled,
          ]}
          onPress={handleAcceptLoad}
          disabled={isAccepting}
        >
          {isAccepting ? (
            <ActivityIndicator size="small" color="#000" />
          ) : (
            <ThemedText style={styles.acceptButtonText}>Accept Load</ThemedText>
          )}
        </Pressable>
      )}
    </View>
  );
}

const styles = StyleSheet.create({
```

## 🧪 ТЕСТИРОВАНИЕ

### На браузере (тестирование ссылки):

1. В PingPoint dashboard создай груз
1. Скопируй **DRIVER MOBILE APP LINK** (должна быть типа `pingpoint://driver/...`)
1. На телефоне отправь себе эту ссылку (через iMessage, SMS, Telegram)
1. Нажми на ссылку
1. Должно открыться приложение PingPoint Driver с деталями груза

### Примеры ссылок которые должны работать:

```
pingpoint://driver/drv_a0a6cf89f39b3b3...
pingpoint://loads/6341a8f5-2193-4bab-8422-9a99762...
https://pingpoint.suverse.io/driver/drv_a0a6cf89f39b3b3...
https://pingpoint.suverse.io/loads/6341a8f5-2193-4bab-8422-9a99762...
```

## 📝 ВАЖНЫЕ МОМЕНТЫ

1. **API Endpoint** - убедись что на production сервере есть endpoint:
   
   ```
   GET /api/loads/:loadId
   ```
   
   Должен возвращать данные груза
1. **Deep Linking iOS** - может потребоваться настройка Associated Domains в Apple
   (уже в app.json, но нужно проверить на production)
1. **Deep Linking Android** - требуется Asset Links JSON файл на сервере
   (уже в app.json через intentFilters)
1. **Тестирование в Expo Go** - deep linking может не работать полностью в Expo Go
   Нужна собранная версия (EAS Build) для полного тестирования
